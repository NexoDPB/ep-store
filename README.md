# E&P Store 🛒

Proyecto web de una tienda virtual de abarrotes desarrollada con HTML, CSS y JavaScript.

## Descripción

E&P Store es una página web responsive que simula una tienda de abarrotes online, donde los usuarios pueden visualizar categorías de productos, ofertas y productos destacados.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript

## Características

- Diseño responsive para dispositivos móviles y escritorio.
- Sección de categorías.
- Carrusel de productos.
- Buscador de productos.
- Diseño moderno y adaptable.
